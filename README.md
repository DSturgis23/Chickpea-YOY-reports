# Chickpea Annual Performance Report

Streamlit web app — annual YOY performance report for Chickpea Pub Group.

## Setup

1. Clone the repo
2. Add credentials to `.streamlit/secrets.toml` (see secrets.toml for template)
3. `pip install -r requirements.txt`
4. `streamlit run app.py`

## Deploy to Streamlit Cloud

Connect GitHub repo at share.streamlit.io
